Racing Game
Este es un simple juego de carreras desarrollado utilizando el framework de juegos Phaser 3. El objetivo del juego es controlar un coche y esquivar obstáculos para ganar puntos. Si el coche choca con un obstáculo, el juego se detiene y puedes reiniciarlo presionando la tecla ESPACIO.

Instrucciones de Uso
1.- Clona este repositorio en tu máquina local o descarga los archivos como un archivo ZIP.
2.- Abre el archivo index.html en tu navegador web para iniciar el juego.
3.- Controla el coche usando las teclas de flecha izquierda (←) y derecha (→).
4.- Esquiva los obstáculos y acumula puntos. Si chocas con un obstáculo, el juego se detendrá y verás un mensaje de "Game Over".
5.- Para reiniciar el juego después de chocar, simplemente presiona la tecla ESPACIO.

Tecnologías Utilizadas
Phaser 3: Un framework de juegos HTML5 utilizado para el desarrollo del juego.
Archivos Principales
index.html: Página HTML que carga Phaser 3 y el archivo del juego.
game.js: Archivo JavaScript que contiene la lógica del juego, incluyendo la creación del coche, obstáculos y detección de colisiones.

Personalización
Si deseas personalizar el juego, puedes editar el archivo game.js para ajustar parámetros como la velocidad del coche, la generación de obstáculos y más.
